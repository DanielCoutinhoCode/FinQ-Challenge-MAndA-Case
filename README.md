# FinQ-Challenge-MAndA-Case
Repositório para publicação de minha análise da Ânima Educação no FinQ M&amp;A Challenge

Visão Geral do Projeto

O objetivo do desafio foi realizar um valuation completo e uma recomendação estratégica para uma transação de M&A real. O trabalho abrange desde o entendimento macroeconômico e setorial até a modelagem detalhada dos fluxos de caixa e sensibilidades de mercado.

Metodologias Utilizadas

Para garantir o rigor analítico, o projeto foi estruturado sobre três pilares de avaliação:
	Fluxo de Caixa Descontado (DCF): Projeção detalhada de Free Cash Flow to the Firm (FCFF), determinação de WACC e cálculo de valor terminal via perpetuidade.
	Múltiplos de Mercado: Comparação relativa utilizando métricas como EV/EBITDA e P/E de empresas comparáveis.
	Transações Precedentes: Análise de deals históricos no setor para balizar o prêmio de controle e as condições de fechamento.

Destaques do Modelo Financeiro

O modelo em Excel foi construído seguindo as melhores práticas de mercado (limpeza de dados, separação de premissas e saídas dinâmicas):
	Análise de Sensibilidade: Matrizes cruzando $WACC$ vs. Crescimento na Perpetuidade (g).
	Cenários Operacionais: Modelagem de cenários (Base, Bull e Bear) impactando as margens operacionais.
	Estrutura de Capital: Avaliação do impacto da alavancagem financeira no valor da firma.
